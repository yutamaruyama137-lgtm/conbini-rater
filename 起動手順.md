# Next.jsアプリの起動手順

## 1. プロジェクトディレクトリに移動

```bash
cd conbini-rater-main
```

**現在のディレクトリ**: `C:\Users\youta\Downloads\conbini-rater-main`  
**移動先**: `C:\Users\youta\Downloads\conbini-rater-main\conbini-rater-main`

## 2. 開発サーバーを起動

```bash
npm run dev
```

または

```bash
npm dev
```

## 3. ブラウザでアクセス

開発サーバーが起動すると、以下のメッセージが表示されます：

```
- ready started server on 0.0.0.0:3000, url: http://localhost:3000
- event compiled client and server successfully
```

ブラウザで以下のURLを開いてください：

**http://localhost:3000**

## トラブルシューティング

### ポート3000が既に使用されている場合

別のポートで起動するには：

```bash
npm run dev -- -p 3001
```

または環境変数で指定：

```bash
PORT=3001 npm run dev
```

### サーバーを停止する場合

ターミナルで `Ctrl + C` を押してください。

### 依存関係がインストールされていない場合

初回起動時や、`node_modules`フォルダがない場合は：

```bash
npm install
```

を実行してから、`npm run dev`を実行してください。
